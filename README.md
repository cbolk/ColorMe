# ColorMe

Progetto per realizzare un dispositivo in grado di rilevare il colore.

## Hardware

+ Sensore TCS3472 (sensore RGB)
+ Seeed Studio Display OLED Grove SSD1306 da 0.66", risoluzione 64×48 monocromatica (visualizzazione colore RGB e hex)
+ ESP32 NodeMCU Modulo WLAN (calcolo e webserver accesso con altri dispositivi)

## Software

+ Gestione sensore
+ Visualizzazione su display
+ Webapp
