# ColorMe -- Versione solo display

## Autore
C. Bolk

## Descrizione

Il sistema integra un sensore di colore (basato su tecnologia a fotodiodi con filtro IR) capace di identificare le componenti cromatiche di una superficie previa calibrazione da parte dell'utente.
Il segnale acquisito viene elaborato dalla scheda per generare coordinate nel modello di colore RGB e la codifca 
esadecimale (es. R: 255 G: 0 B: 0 hex: #FF0000).

Il risultato viene visualizzato sul display.
